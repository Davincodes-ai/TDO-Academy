T.D.O Academy Website 🌟
Welcome to the T.D.O Academy website repository! This project serves as the foundation of our online learning platform, designed to empower aspiring coders with accessible, interactive, and engaging coding education.

Features 🚀
Home Page: A welcoming introduction to T.D.O Academy.
About Us: Insights into our mission, vision, and team.
Courses Page: Explore our diverse coding courses with detailed descriptions.
User-Friendly Design: Fully responsive and optimized for mobile, tablet, and desktop.
Interactive Components: Engage users with quizzes, blogs, and resources.
Technologies Used 🛠️
Front-End: HTML, CSS, JavaScript.
Back-End: Python (Flask Framework).
Styling: CSS and Bootstrap for responsiveness.
Scripts: JavaScript for interactivity.
Hosting: Flask development server (local) or production hosting.
